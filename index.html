CodeCademy Cheat Sheet

HTML

Elements and Structure
Tag Name Description
<body> Body Represents the content of an HTML document.
<h1> - <h6> Headings The heading elements are used to highlight the headers and go from h1 to h6 that represent the order of
importance due to hierarchy.
<div> Division The div element is used as a containe that divides an HTML document in sections; it can contain flow
content such as headings, paragraphs, links, images, etc.
<p> Paragraph The paragraph element and displays a block of text
<span> Span The span element is an inline container for text and can be used to group text for styling purposes.
<strong> Strong The strong element highlights important, serious, or urgent text and browsers will normally render this
highlighted text in bold by default.
<br< Line Break The line break element will create a line break in text and is especially useful where a division of text
is required. It is a self-closing tag.
<ul> Unordered List The unordered list element is used to create a list of items in no particular order.
<ol> Ordered List The ordered list element creates a list of items in sequential order.
<li> List Item The list item element create list items inside unordered and ordered lists.
<img> Image HTML image elements embed images in documents.
<video> Video The video element embeds a media player for video playback.
<html> HTML The HTML element, the root of an HTML document, should be added after the !DOCTYPE declaration.
<head> Head The head element contains general information about an HTML page that isn’t displayed on the page
itself. This information is called metadata and includes things like the title of the HTML document
and links to stylesheets.
<title> Title The title element contains a text that defines the title of an HTML document. The title is displayed in the
browser’s title bar or tab in which the HTML page is displayed.
<a> Anchor The anchor element is used to create hyperlinks in an HTML document.
<!-- Comment --> Comments Content inside of comments will not be rendered by browsers, and are usually used to describe a part of
code or provide other details.

Command Line

Navigating the Command Line
Command Name Description
ls List Is used to list the contents of directories.
pwd Print Working Directory Displays the file path from the root directory to the current working directory.
cd Change Directory Is used to move throughout the filesystem of a computer.
mkdir Make Directory Is used to move throughout the filesystem of a computer.
touch Create New File Creates a new file in the current working directory with the name provided.
